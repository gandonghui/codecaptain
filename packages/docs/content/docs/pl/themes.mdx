# Motywy

CodeCaptain obsługuje wbudowane motywy oraz własne pliki JSON z motywami.

## Dodaj własny motyw

1. Utwórz katalog motywów:

```bash
mkdir -p ~/.config/codecaptain/themes
```

2. Dodaj swój plik JSON do tego katalogu (na przykład `my-theme.json`).
3. Otwórz CodeCaptain, a następnie przejdź do **Settings -> Theme -> Reload themes**.
4. Wybierz swój motyw z listy rozwijanej. Zostanie zastosowany od razu.

## Lokalizacja motywów

- macOS/Linux: `~/.config/codecaptain/themes/`

## Pełna dokumentacja formatu JSON

Skorzystaj z pełnego przewodnika po formacie w dokumentacji głównego repozytorium:

- [`docs/CUSTOM_THEMES.md`](https://github.com/codecaptain/codecaptain/blob/main/docs/CUSTOM_THEMES.md)
