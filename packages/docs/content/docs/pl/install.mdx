# Instalacja

CodeCaptain można uruchomić na trzy sposoby:

- aplikacja na komputer dla macOS
- aplikacja webowa hostowana przez CLI, którą możesz zainstalować jak aplikację na telefonie (jako PWA)
- rozszerzenie do VS Code

## Wymaganie wstępne

Najpierw zainstaluj [CodeCaptain-core](https://opencode.ai) — CodeCaptain działa na jego bazie.

## Przeglądarka + PWA

```bash
curl -fsSL https://raw.githubusercontent.com/codecaptain/codecaptain/main/scripts/install.sh | bash
codecaptain --ui-password be-creative-here
```

Otwórz adres URL wypisany przez CLI (zwykle `http://localhost:3000`). Powinna pojawić się lista sesji CodeCaptain. Aby mieć ją zawsze pod ręką, użyj opcji „Zainstaluj” w pasku adresu przeglądarki, aby dodać ją jako aplikację.

## Komputer

Pobierz najnowszą kompilację na komputer ze strony wydań na GitHubie lub ze strony pobierania CodeCaptain. Otwórz ją i zaloguj się do swojego zwykłego procesu pracy CodeCaptain-core.

## VS Code

Zainstaluj z VS Code Marketplace i zaloguj się do swojego zwykłego procesu pracy CodeCaptain-core. Widok CodeCaptain otworzy się wtedy na pasku bocznym.
