# Instalação

Há três formas de rodar o CodeCaptain:

- app de desktop para macOS
- app web hospedado pela CLI, que você pode instalar como um app de celular (um PWA)
- extensão para VS Code

## Pré-requisito

Instale o [CodeCaptain-core](https://opencode.ai) primeiro — o CodeCaptain roda em cima dele.

## Web + PWA

```bash
curl -fsSL https://raw.githubusercontent.com/codecaptain/codecaptain/main/scripts/install.sh | bash
codecaptain --ui-password be-creative-here
```

Abra a URL que a CLI imprime (geralmente `http://localhost:3000`). Você deve ver a lista de sessões do CodeCaptain. Para deixá-lo à mão, use a opção "Instalar" do seu navegador na barra de endereço para adicioná-lo como um app.

## Desktop

Baixe a build de desktop mais recente na página de releases do GitHub ou na página de download do CodeCaptain. Abra-a e entre no seu fluxo de trabalho habitual do CodeCaptain-core.

## VS Code

Instale pela VS Code Marketplace e entre no seu fluxo de trabalho habitual do CodeCaptain-core. A visualização do CodeCaptain então abre na barra lateral.
