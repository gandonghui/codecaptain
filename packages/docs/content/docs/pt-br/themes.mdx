# Temas

O CodeCaptain oferece suporte a temas integrados e arquivos JSON de tema personalizados.

## Adicionar um tema personalizado

1. Crie o diretório de temas:

```bash
mkdir -p ~/.config/codecaptain/themes
```

2. Adicione seu arquivo JSON a esse diretório (por exemplo, `my-theme.json`).
3. Abra o CodeCaptain e vá em **Settings -> Theme -> Reload themes**.
4. Escolha seu tema na lista suspensa. Ele é aplicado na hora.

## Localização dos temas

- macOS/Linux: `~/.config/codecaptain/themes/`
