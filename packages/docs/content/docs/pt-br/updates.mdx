# Atualizações

Como você atualiza o CodeCaptain depende de como o instalou. Em todos os casos, você pode verificar sua versão atual em **Settings → CodeCaptain → About**.

## App de desktop

O app de desktop verifica atualizações nas releases do GitHub. Quando há uma disponível, o CodeCaptain avisa você, baixa-a quando você escolhe e a instala no próximo reinício. Você está sempre no controle — nada é instalado sem a sua autorização.

## Web / CLI

Se você instalou a CLI, atualize-a a partir de **About** com os botões de verificar e atualizar, ou pelo terminal:

```bash
codecaptain update
```

O CodeCaptain detecta como foi instalado (npm, pnpm, yarn ou bun) e executa a atualização certa para você.

## O servidor CodeCaptain-core

O CodeCaptain e o CodeCaptain-core são atualizados separadamente. Quando uma nova versão do CodeCaptain-core está disponível, o CodeCaptain oferece atualizá-la e reinicia o servidor em seguida. Veja [CodeCaptain-core Server](/pt-br/opencode-server/) para como esse servidor é gerenciado.
