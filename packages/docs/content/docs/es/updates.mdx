# Actualizaciones

Cómo actualizas CodeCaptain depende de cómo lo instalaste. En todos los casos puedes comprobar tu versión actual en **Settings → CodeCaptain → About**.

## App de escritorio

La app de escritorio busca actualizaciones contra las releases de GitHub. Cuando hay una disponible, CodeCaptain te avisa, la descarga cuando lo eliges y la instala en el siguiente reinicio. Siempre tienes el control: nada se instala sin tu aprobación.

## Web / CLI

Si instalaste la CLI, actualízala desde **About** con los botones de comprobar y actualizar, o desde la terminal:

```bash
codecaptain update
```

CodeCaptain detecta cómo se instaló (npm, pnpm, yarn o bun) y ejecuta la actualización correcta por ti.

## El servidor de CodeCaptain-core

CodeCaptain y CodeCaptain-core se actualizan por separado. Cuando hay una nueva versión de CodeCaptain-core disponible, CodeCaptain se ofrece a actualizarla y reinicia el servidor después. Consulta [CodeCaptain-core Server](/es/opencode-server/) para saber cómo se gestiona ese servidor.
