# Instalación

Hay tres formas de ejecutar CodeCaptain:

- app de escritorio para macOS
- app web alojada por la CLI, que puedes instalar como una app de teléfono (una PWA)
- extensión para VS Code

## Requisito previo

Instala [CodeCaptain-core](https://opencode.ai) primero: CodeCaptain se ejecuta sobre él.

## Web + PWA

```bash
curl -fsSL https://raw.githubusercontent.com/codecaptain/codecaptain/main/scripts/install.sh | bash
codecaptain --ui-password be-creative-here
```

Abre la URL que imprime la CLI (normalmente `http://localhost:3000`). Deberías ver la lista de sesiones de CodeCaptain. Para tenerlo a mano, usa la opción "Instalar" de tu navegador en la barra de direcciones para añadirlo como una app.

## Escritorio

Descarga la última compilación de escritorio desde la página de releases de GitHub o la página de descargas de CodeCaptain. Ábrela e inicia sesión con tu flujo de trabajo habitual de CodeCaptain-core.

## VS Code

Instálala desde el VS Code Marketplace e inicia sesión con tu flujo de trabajo habitual de CodeCaptain-core. La vista de CodeCaptain se abre entonces en la barra lateral.
