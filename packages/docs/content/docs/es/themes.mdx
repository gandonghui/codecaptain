# Temas

CodeCaptain admite temas integrados y archivos JSON de tema personalizados.

## Añadir un tema personalizado

1. Crea el directorio de temas:

```bash
mkdir -p ~/.config/codecaptain/themes
```

2. Añade tu archivo JSON a ese directorio (por ejemplo, `my-theme.json`).
3. Abre CodeCaptain y ve a **Settings -> Theme -> Reload themes**.
4. Elige tu tema en el desplegable. Se aplica de inmediato.

## Ubicación de los temas

- macOS/Linux: `~/.config/codecaptain/themes/`
