# Теми

CodeCaptain підтримує вбудовані теми та власні JSON-файли тем.

## Додати власну тему

1. Створіть директорію тем:

```bash
mkdir -p ~/.config/codecaptain/themes
```

2. Додайте свій JSON-файл у цю директорію (наприклад, `my-theme.json`).
3. Відкрийте CodeCaptain, потім перейдіть до **Settings -> Theme -> Reload themes**.
4. Оберіть свою тему зі спадного списку. Вона застосується одразу.

## Розташування тем

- macOS/Linux: `~/.config/codecaptain/themes/`

## Повний довідник формату JSON

Скористайтеся повним посібником із формату в доках основного репозиторію:

- [`docs/CUSTOM_THEMES.md`](https://github.com/codecaptain/codecaptain/blob/main/docs/CUSTOM_THEMES.md)
