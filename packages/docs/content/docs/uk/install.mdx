# Встановлення

Є три способи запустити CodeCaptain:

- десктоп-застосунок для macOS
- вебзастосунок на базі CLI, який можна встановити як застосунок на телефоні (PWA)
- розширення для VS Code

## Передумова

Спершу встановіть [CodeCaptain-core](https://opencode.ai) — CodeCaptain працює поверх нього.

## Веб + PWA

```bash
curl -fsSL https://raw.githubusercontent.com/codecaptain/codecaptain/main/scripts/install.sh | bash
codecaptain --ui-password be-creative-here
```

Відкрийте URL, який вивів CLI (зазвичай `http://localhost:3000`). Ви маєте побачити список сесій CodeCaptain. Щоб він був під рукою, скористайтеся опцією «Install» в адресному рядку браузера й додайте його як застосунок.

## Десктоп

Завантажте найновішу десктопну збірку зі сторінки релізів на GitHub або зі сторінки завантаження CodeCaptain. Відкрийте її та увійдіть у свій звичний робочий процес CodeCaptain-core.

## VS Code

Встановіть із VS Code Marketplace і увійдіть у свій звичний робочий процес CodeCaptain-core. Після цього вікно CodeCaptain відкриється на бічній панелі.
