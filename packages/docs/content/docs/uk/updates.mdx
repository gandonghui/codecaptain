# Оновлення

Те, як ви оновлюєте CodeCaptain, залежить від того, як ви його встановили. У будь-якому разі ви можете перевірити свою поточну версію в **Settings → CodeCaptain → About**.

## Десктопний застосунок

Десктопний застосунок перевіряє наявність оновлень у релізах GitHub. Коли воно доступне, CodeCaptain повідомляє вам, завантажує його, коли ви оберете, і встановлює при наступному перезапуску. Ви завжди керуєте процесом — нічого не встановлюється без вашого схвалення.

## Веб / CLI

Якщо ви встановили CLI, оновіть його з **About** кнопками перевірки та оновлення або з терміналу:

```bash
codecaptain update
```

CodeCaptain визначає, як його було встановлено (npm, pnpm, yarn чи bun), і запускає правильне оновлення за вас.

## Сервер CodeCaptain-core

CodeCaptain та CodeCaptain-core оновлюються окремо. Коли доступна нова версія CodeCaptain-core, CodeCaptain пропонує оновити її й перезапускає сервер після цього. Див. [CodeCaptain-core Server](/uk/opencode-server/) про те, як цим сервером керують.
