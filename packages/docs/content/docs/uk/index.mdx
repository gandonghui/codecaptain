# Документація CodeCaptain

CodeCaptain — це візуальний робочий простір навколо CodeCaptain-core (AI-агента для кодингу, який працює у вашому терміналі). Він дає екран, де можна спостерігати за цією роботою та керувати нею, замість того щоб лишатися в командному рядку.

Використовуйте ці доки, щоб:

- встановити застосунок, що пасує вашому робочому процесу
- безпечно відкрити CodeCaptain для віддаленого використання
- налаштувати вигляд і усувати типові проблеми

## Для чого CodeCaptain

CodeCaptain створений для тих частин AI-кодингу, яким потрібен командний центр: розгалуження сесій, перегляд діфів, керування терміналами, спостереження за прогресом інструментів, запуск дій проєкту та збереження всієї картини перед очима, поки агент працює.
