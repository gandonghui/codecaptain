# Themes

CodeCaptain supports built-in themes and custom theme JSON files.

## Add a custom theme

1. Create the themes directory:

```bash
mkdir -p ~/.config/codecaptain/themes
```

2. Add your JSON file to that directory (for example `my-theme.json`).
3. Open CodeCaptain, then go to **Settings -> Theme -> Reload themes**.
4. Pick your theme from the dropdown. It applies right away.

## Theme location

- macOS/Linux: `~/.config/codecaptain/themes/`
