# 更新

你如何更新 CodeCaptain 取决于你如何安装它。在任何情况下，你都可以在 **Settings → CodeCaptain → About** 查看你的当前版本。

## 桌面应用

桌面应用会对照 GitHub 发布版本检查更新。当有可用更新时，CodeCaptain 会告诉你，在你选择时下载它，并在下次重启时安装它。一切都由你掌控 — 没有你的同意不会安装任何东西。

## 网页端 / CLI

如果你安装的是 CLI，可以从 **About** 用检查和更新按钮更新它，或从终端更新：

```bash
codecaptain update
```

CodeCaptain 会检测它是如何安装的（npm、pnpm、yarn 或 bun），并为你运行正确的更新。

## CodeCaptain-core 服务器

CodeCaptain 和 CodeCaptain-core 分别更新。当有新的 CodeCaptain-core 版本可用时，CodeCaptain 会提议更新它，并在之后重启服务器。关于该服务器如何被管理，请参阅 [CodeCaptain-core Server](/zh-cn/opencode-server/)。
