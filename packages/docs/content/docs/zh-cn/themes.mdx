# 主题

CodeCaptain 支持内置主题和自定义主题 JSON 文件。

## 添加自定义主题

1. 创建主题目录：

```bash
mkdir -p ~/.config/codecaptain/themes
```

2. 将你的 JSON 文件添加到该目录（例如 `my-theme.json`）。
3. 打开 CodeCaptain，然后前往 **Settings -> Theme -> Reload themes**。
4. 从下拉菜单中选择你的主题。它会立即生效。

## 主题位置

- macOS/Linux：`~/.config/codecaptain/themes/`
