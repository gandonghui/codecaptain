# Install

There are three ways to run CodeCaptain:

- desktop app for macOS
- web app hosted by the CLI, which you can install like a phone app (a PWA)
- VS Code extension

## Prerequisite

Install [CodeCaptain-core](https://opencode.ai) first — CodeCaptain runs on top of it.

## Web + PWA

```bash
curl -fsSL https://raw.githubusercontent.com/codecaptain/codecaptain/main/scripts/install.sh | bash
codecaptain --ui-password be-creative-here
```

Open the URL the CLI prints (usually `http://localhost:3000`). You should see the CodeCaptain session list. To keep it handy, use your browser's "Install" option in the address bar to add it as an app.

## Desktop

Download the latest desktop build from the GitHub releases page or the CodeCaptain download page. Open it and sign into your usual CodeCaptain-core workflow.

## VS Code

Install from the VS Code Marketplace and sign into your usual CodeCaptain-core workflow. The CodeCaptain view then opens in the sidebar.
