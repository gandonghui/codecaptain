# CodeCaptain Docs

CodeCaptain is the visual workspace around CodeCaptain-core (the AI coding agent that runs in your terminal). It gives you a screen to watch and steer that work, instead of living on the command line.

Use these docs to:

- install the right app for how you work
- open CodeCaptain for safe remote use
- customize the look and fix common issues

## What CodeCaptain is for

CodeCaptain is for the parts of AI coding that benefit from a control room: branching sessions, reviewing diffs, managing terminals, watching tool progress, running project actions, and keeping the full board visible while the agent works.
