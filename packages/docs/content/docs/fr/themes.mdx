# Thèmes

CodeCaptain prend en charge les thèmes intégrés et les fichiers JSON de thème personnalisés.

## Ajouter un thème personnalisé

1. Créez le dossier des thèmes :

```bash
mkdir -p ~/.config/codecaptain/themes
```

2. Ajoutez votre fichier JSON dans ce dossier (par exemple `my-theme.json`).
3. Ouvrez CodeCaptain, puis allez dans **Paramètres -> Thème -> Reload themes**.
4. Choisissez votre thème dans le menu déroulant. Il s’applique immédiatement.

## Emplacement des thèmes

- macOS/Linux : `~/.config/codecaptain/themes/`
