# Installation

Il existe trois façons d’utiliser CodeCaptain :

- l’application desktop pour macOS
- l’application web servie par la CLI, que vous pouvez installer comme une application de téléphone (une PWA)
- l’extension VS Code

## Prérequis

Installez d’abord [CodeCaptain-core](https://opencode.ai) — CodeCaptain fonctionne par-dessus.

## Web + PWA

```bash
curl -fsSL https://raw.githubusercontent.com/codecaptain/codecaptain/main/scripts/install.sh | bash
codecaptain --ui-password be-creative-here
```

Ouvrez l’URL affichée par la CLI (souvent `http://localhost:3000`). Vous devriez voir la liste des sessions CodeCaptain. Pour la garder à portée de main, utilisez l’option « Installer » de votre navigateur dans la barre d’adresse pour l’ajouter comme application.

## Desktop

Téléchargez la dernière version desktop depuis la page des releases GitHub ou la page de téléchargement CodeCaptain. Ouvrez-la et connectez-la à votre workflow CodeCaptain-core habituel.

## VS Code

Installez l’extension depuis le VS Code Marketplace et connectez-la à votre workflow CodeCaptain-core habituel. La vue CodeCaptain s’ouvre ensuite dans la barre latérale.
